# Mensa CLI

## Usage

```sh
# get todays dishes
./mensa.py
# get dishes for tomorrow
./mensa.py morgen
# get dishes for a specific weekday
./mensa.py donnerstag
```

## Installation

Install requirements using apt:

    apt install python3-bs4 python3-requests

Or install requirements using pip:

    pip install beautifulsoup4 requests
